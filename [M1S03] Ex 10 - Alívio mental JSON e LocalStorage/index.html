Ex 10 - Alívio mental: JSON e LocalStorage

Chegamos ao fim! Para aliviar a cabeça, vamos fazer um exercício simples.

Explique o que é um JSON e como funcionam os métodos JSON.parse e JSON.stringify.

JSON:

JSON significa "JavaScript Object Notation", é um formato simples e leve para troca de dados entre sistemas, a maioria das linguagens atuais possuem suporte a este formato.

JSON.parse:

Transforma uma string para objeto. Essa string precisa estar no formato chave/valor entre chaves. Exemplo {"result":true, "count":42}

Exemplo:

const json = '{"result":true, "count":42}'
const obj = JSON.parse(json)
console.log(obj.count)

No exemplo acima sera exibido o valor 42

JSON.stringify:

Converte valores para uma notação JSON

Exemplo:

const myArr = ['bacon', 'lettuce', 'tomatoes']
const myArrStr = JSON.stringify(myArr)
console.log(myArrStr)

No exemplo acima sera exibido o valor "["bacon","lettuce","tomatoes"]"
